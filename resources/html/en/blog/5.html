Why GNU/Linux with KDE Plasma is my favorite operating system

If you know nothing about Linux…


I have no intention in elaborate a tutorial about Linux in this publication, but if you know nothing about Linux, here is a rough explanation:

Linux is an operating system like Windows and macOS (actually, Linux is just the system's kernel, but I will not enter these technicalities…) that falls in the classification of free software, it means that anyone has access to the source code, and they can launch their own modifications, that's why different organizations launch their “Linux distributions”, that are Linux operating systems with their particularities. Even you, who is reading this text, can create your own Linux distribution if you have enough knowledge and will. Here are the names of some of the famous distributions: Ubuntu, Debian, Fedora, Arch Linux, Linux Mint, openSUSE, Mageia, etc.

It's a very modular operating system, it can have different init systems, graphics servers, sounds servers, graphical interfaces with their respective standard applications, etc. The most famous graphical interfaces are GNOME, KDE Plasma, XFCE, Cinnamon, LXQt, MATE, i3, Pantheon, and Budgie. My favorite one is KDE Plasma, in second place is Cinnamon.

I will not extend myself here, I just want you to know that the distributions are really different. I like Linux with KDE Plasma, I can have this combination through Fedora, Kubuntu, Debian, openSUSE, Mageia, Arch Linux, and many other distributions, and even so the experience will be different.


Now allow me to indeed start the post


I don't know how I did hear about GNU/Linux for the first time, but my first distribution was the nostalgic Kurumin 7 (a Brazilian distribution) at the end of the 2000s. I will never forget when I saw that beautiful and colorful interface for the first time, and the fact that the operating system already had several programs installed by default, what is different from Windows XP that came “raw”. From now on, I will only use “Linux” to refer to the operating system as a whole.


Also, it was in that epoch that I knew the free software movement, that I got identified. From that moment, I always sought to use free software, but I never became a radical, and I didn't completely abandoned Windows because of the problematic hardware of my laptop (SiS graphic chipset…)

Beyond the better security, Linux changed the way I use a operating system: I no more fell the need of using pirate software, all of its graphical environments incentive me to not put icons in the desktop (what gives more empty space for showing the wallpaper), I can insert characters like “ĵ”, “ŭ”, “ŕ”, “ḯ”, etc without the need of a additional program (what is convenient for the ones who like to write in other languages like Esperanto), I can manipulate files – including for deleting them – even if they are opened by other programs, its performance is better than Windows, it offers more privacy, and – here comes the icing on the cake – it is a good operating system for programming.

But the biggest advantage is in the graphical environments, more precisely in KDE Plasma. I started writing this post in October 2022 and, up to this date, Windows Explorer still didn't have support for tabs, I repeat, the file manager of Windows didn't have support for tabs, something that file managers of Linux had for more than a decade! Actually, many new features come to Linux world first, like a native system of automatic change of wallpapers, or automatic change of sound profile when you are using the computer's speakers or earphones (yes, Windows already has it, but these features reached to Linux first).

Ok, but how KDE Plasma (and Cinnamon too) stand out in relation to the other graphical environments?

  • Native function for deactivating the laptop's touchpad when the USB mouse is connected. It is very useful when the touchpad is too big…
  • Capability to habilitate the configuration of individual exhibition of files and folders. Didn't you understand? For example: in GNOME and XFCE, if I configure the file manager to show the files and folders in alphabetical order and in list format, this change will be applied to all folders; but in KDE Plasma and Cinnamon, I can make this configuration to be applied to a single folder, I can let in this way in my folder of music, but for the rest of my folders I let the files and folders to be shown in big icon format and in chronological order.
