Porque GNU/Linux com KDE Plasma é o meu sistema operacional favorito

Se você não sabe nada de Linux…


Não tenho a menor intenção de elaborar um tutorial sobre Linux nesta publicação, mas se você não entende nada de Linux, aqui vai uma explicação grosseira:

Linux é um sistema operacional como Windows e macOS (na verdade, Linux é só o kernel do sistema, mas não entrarei nessas tecnicidades…) que cai na classificação de software livre, isso significa que qualquer um tem acesso ao seu código fonte e pode lançar suas próprias modificações, por isso que diferentes organizações lançam as suas “distribuições Linux”, que são sistemas operacionais Linux com suas particularidades. Até mesmo você, que está lendo este texto, pode criar sua própria distribuição Linux se você tiver conhecimento e vontade suficientes. Eis os nomes de algumas distribuições famosas: Ubuntu, Debian, Fedora, Arch Linux, Linux Mint, openSUSE, Mageia, etc.

É um sistema operacional bem modular, podendo ter até mesmo diferentes sistemas de inicialização, servidores gráficos, servidores de som, interfaces gráficas com seus respectivos aplicativos padrão, etc. As interfaces gráficas mais famosas são GNOME, KDE Plasma, XFCE, Cinnamon, LXQt, MATE, i3, Pantheon e Budgie. A minha preferida é o KDE Plasma, em segundo lugar está o Cinnamon.

Não me estenderei mais aqui, só quero que você saiba que as distribuições realmente são diferentes. Eu gosto do Linux com KDE Plasma, posso ter essa combinação no Fedora, Kubuntu, Debian, openSUSE, Mageia, Arch Linux e muitas outras distribuições, e mesmo assim a experiência poderá ser diferente.


Agora permita-me iniciar o post de fato


Não sei como eu ouvi falar do GNU/Linux pela primeira vez, mas minha primeira distribuição foi o saudoso Kurumin 7 no final da década de 2000. Nunca me esquecerei de quando vi aquela bela e colorida interface pela primeira vez e do o fato de que o sistema operacional já continha vários programas instalados por padrão, diferente do Windows XP que vinha “cru”. Daqui em diante usarei apenas “Linux” para me referir ao sistema operacional como um todo.


Também foi nessa época que conheci o movimento do software livre, com o qual me identifiquei. A partir desse momento, eu sempre procurei usar software livre, mas nunca cheguei a ser um radical e não abandonei o Windows completamente por conta do harware problemático do meu notebook (chipset gráfico SiS…).

Além de uma maior segurança, Linux mudou meu uso de um sistema operacional: não senti mais a necessidade de usar software pirata, eu posso inserir caracteres como “ĵ”, “ŭ”, “ŕ”, “ḯ”, etc sem a necessidade de software adicional (o que é muito conveniente para quem gosta de escrever em outras línguas como esperanto), posso manipular arquivos – inclusive deletá-los – mesmo que eles estejam abertos por outros programas, deixa gravados os conteúdos anteriormente guardados na área de transferência, é mais rápido que o Windows, oferece maior privacidade e – aqui vem a cereja do bolo – é um bom sistema operacional para programar.

Mas a vantagem maior está nos ambientes gráficos, mais precisamente no KDE Plasma. Comecei a escrever este post em outubro de 2022 e, até esta data, o Windows Explorer não tinha suporte a abas, repito, o gerenciador de arquivos do Windows não tinha suporte a abas, algo que os gerenciadores de arquivos do Linux já tinham há mais de uma década! Na verdade, muitas das novidades chegam no mundo Linux primeiro, como um sistema nativo de mudança automática de papel de parede ou mudança automática de perfil de som quando se usa as saídas de som do computador ou fones de ouvido (sim, Windows já tem tudo isso, mas essas features chegaram no Linux primeiro).

Ok, mas como o KDE Plasma (e também o Cinnamon) se destacam em relação aos outros ambientes gráficos?

  • Função nativa de desativar o touchpad do notebook quando o mouse USB é conectado. Muito útil quando o touchpad é muito grande…
  • Capacidade de habilitar a configuração de exibição individual de arquivos e pastas. Não entendeu? Por exemplo: no GNOME ou XFCE, se configuro para que o gerenciador de arquivos exiba os arquivos e pastas de forma alfabética e em formato de lista, essa mudança será aplicada a todas as pastas; já no KDE Plasma e Cinnamon, posso fazer com que essa configuração seja aplicada a uma única pasta, posso deixar assim na minha pasta de música, mas para o resto eu posso deixar que os arquivos e pastas sejam exibidos em formato de ícones grandes e em ordem cronológica.
